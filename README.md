# BMP CLI Editor

Консольная утилита на C++ для обработки **24-битных BMP-изображений без сжатия**. Поддерживает вывод информации о файле, рисование квадрата с диагоналями, окружности, RGB-фильтр, поворот прямоугольной области и добавление внешней рамки.

Было выполнено в рамках курсовой работы 2 курса.

## Запуск

## 1. Обычный локальный запуск

### Зависимости

Для Ubuntu / Debian:

```bash
sudo apt-get update
sudo apt-get install -y g++ make libc6-dev libstdc++6
```

### Сборка

```bash
cd src
make
```

После сборки появится исполняемый файл `cw`.

### Запуск

```bash
./cw --help
```

---

## 2. Docker

### Скачать готовый образ

```bash
sudo docker pull ghcr.io/mihailmurdasov/bmp-cli-editor:1.0.0
```

### Запуск

```bash
sudo docker run --rm -it -v "$PWD":/work -w /work ghcr.io/mihailmurdasov/bmp-cli-editor:1.0.0 <аргументы_программы>
```

## 3. Скрипт

### Сделать скрипт исполняемым


```bash
./cw-docker <аргументы_программы>
```

## О проекте

Поддерживаемые операции:

- `--info` — вывод заголовков BMP-файла
- `--squared_lines` — рисование квадрата с диагоналями
- `--rgbfilter` — замена значения одной RGB-компоненты
- `--rotate` — поворот прямоугольной области на `90`, `180`, `270` градусов
- `--circle` — рисование окружности с опциональной заливкой
- `--outside_ornament` — добавление внешней цветной рамки

## Требования к входным данным

Утилита работает только с BMP-файлами, которые одновременно удовлетворяют условиям:

- формат `BMP`
- без сжатия
- `24 bits per pixel`

Если выходной файл не указан, результат по умолчанию сохраняется как `out.bmp`. Для операции `--info` выходной файл не требуется.

## Коды выхода

- `0` — успешное завершение
- `40` — ошибка чтения файла
- `41` — ошибка записи файла
- `42` — некорректный BMP-файл
- `43` — некорректный параметр
- `44` — некорректный аргумент
- `45` — некорректное завершение работы

## Используемые технологии

- **C++11**
- стандартная библиотека C++
- `getopt_long` для разбора аргументов командной строки
- работа с бинарными файлами через `fstream`
- ручной разбор заголовков BMP
- `Makefile` для сборки
- Docker / GHCR для воспроизводимого запуска и распространения готового image
